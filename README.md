8085 Microprocessor-Based Computer System

This project implements a complete 8085 microprocessor system designed using digital logic components and memory interfacing techniques. The system includes ROM and RAM integration, address decoding, input/output peripherals, clock generation, and reset circuit.

Features
Intel 8085 CPU running at 6 MHz
Address decoding using 74LS138
Two EPROM chips for program memory
Two SRAM chips for data memory
Memory-mapped 7-segment display output
8-bit DIP switch input through a 74LS244 buffer
Reset and clock generation circuits
Logic gate implementation using 74LS-series ICs
Complete schematic designed for simulation and hardware implementation

Components Used
Intel 8085 Microprocessor
74LS138 Address Decoder
74LS244 Octal Buffer
HM6264 SRAM
EPROM Memory ICs
Common Anode 7-Segment Display
DIP Switches
Crystal Oscillator (6 MHz)
TTL Logic Gates (74LS08 and others)

Purpose
This project demonstrates the fundamentals of microprocessor system design, including memory interfacing, address mapping, peripheral communication, and hardware integration.
